24/7 Reparatur-Service

Anleitung

Vielen Dank, dass Sie sich dazu entschieden haben Ihren Computer in die ArctusLabs-Repairs zu geben. Auf dieser Seite finden Sie eine ausführliche Anleitung zur Vorbereitung, Verpackung und Versendung Ihres PCs.

Schritt 1: Vorbereitung

Zur Vorbereitung benötigen Sie folgendes:

  • Eine stabile Verpackung mit entsprechender Größe (z.B. Karton).
    Über den folgenden Link können Sie Verpackungs- und Versandmaterial online und kostengünstig bestellen, falls Sie keine Verpackung zur Hand haben: https://www.karton.eu/Verpackungsmaterial
  • Ein Paketband
  • Eine Kopie des Paketscheins, die Sie auf das Paket kleben können
  • Notiz mit entsprechenden Daten (siehe unten) und Schilderung des Problems bzw. des Defekts
Bitte stellen Sie zudem sicher, dass Sie alle wichtigen Daten auf einem externen Speichermedium gesichert haben. Bitte beachten Sie auch, dass wir keine Haftung für Datenverlust übernehmen können.

Schritt 2: Verpackung

Bitte verpacken Sie Ihren Computer so, dass er sicher und unbeschädigt bei uns ankommt. Falls kein Problem auf den Festplatten besteht, so können Sie diese auch entnehmen. Verpacken Sie Ihren Computer nun so, dass dieser vor Stößen und Beschädigungen geschützt ist. Für zusätzlichen Schutz können Sie den Karton mit Füllmaterial befüllen. Die Grafikkarte sollte zudem zuvor abgebaut und sicher verpackt werden, damit keine Schäden entstehen und beachten Sie die zusätzlichen Regelungen zur Verpackung von Gegenstände mit Sonderbehandlung und von verbotene Gegenstände. Bitte beachten Sie zusätzlich, dass wir keine Haftung für Lieferbeschädigungen aller Art übernehmen können.

Für einen reibungslosen Verlauf: Bitte formulieren Sie das Problem in die entsprechenden Eingabefelder. Sollte dies nicht möglich sein, so formulieren Sie das Problem bitte in leserlicher Handschrift und fügen Sie diese dem Paket bei.
Zum Schluss kleben Sie bitte den Aufdruck gut sichtbar auf das Paket. Haben Sie Probleme, beim Verpacken Ihres Computers? Hier finden Sie zusätzliche Hilfe: http://www.fedex.com/downloads/howtopack/HowToPack_EMEA_DEDE.pdf

Schritt 3: Versand

Wählen Sie aus, mit welcher Versandart Sie Ihren Computer zu uns senden möchten.

  • Per DHL: https://www.dhl.de/de/privatkunden/pakete-versenden/deutschlandweit-versenden/preise-national.html
  • Per Hermes: https://www.myhermes.de/preise/paeckchen-paket/
  • Per dpd: https://my.dpd.de/sending/sendparcel.aspx
  • Per UPS: https://www.ups.com/de/de/shipping/how-to-ship-package.page

    • Bitte senden Sie Ihren Computer an die folgende Adresse:
    ArctusLabs GmbH
    Höffgeshofweg 26A
    47807 Krefeld
    Deutschland
    Bitte beachten Sie, dass wir keine Haftung für Lieferbeschädigungen aller Art übernehmen können.
